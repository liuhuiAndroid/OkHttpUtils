# OkHttpUtils
okhttp的辅助类
- 借鉴了各种封装okhttp项目，为了学习原理并且方便日后项目中使用。如有侵权，请联系本人删除。
# 功能
- 一般的get请求
- 一般的post请求
- 基于Http Post的文件上传
- 文件下载
